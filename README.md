## Custom Data Structures & Algorithms Library (Java)

### Overview
A Java-based implementation of core data structures and algorithms built from scratch.

### Implemented Structures
- Dynamic Array
- Linked List
- Stack
- Queue
- HashMap (Chaining)
- Binary Search Tree
- Graph (BFS & DFS)

### Concepts Demonstrated
- Data structures implementation
- Algorithm development
- Object-oriented programming
- Time and space complexity awareness

### How to Run
Compile and run Main.java
